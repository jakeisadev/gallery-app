# gallery-app

I really enjoyed making this project.

Admittedly, I had to hire a tutor to help me learn how to call the API and also how to get the modals to work properly. With that said, I was able to familiarize myself with the structure of Bootstrap and dynamically inserting HTML elements via JS loops on my own. It felt great to finally have enough knowledge and confidence to take that risk of implementing, and even more gratifying to lessen HTML nodes in index.html for increased readability.

The most challenging part of this project was certainly learning how to download each picture once the user entered the modals. I also required third party guidance on how to implement that. What is important, is that I insisted each step was broken down to me so I could clearly understand the why behind the way each part of the program worked. 

I came away with a solid understanding of substrings for when I had to program the image path to make it accessible to downnload, how to dynamically edit modals that are were already dynamically programmed as HTML (I had to learn how to access the data fetch loop from the openImageModal loop. Or more accurately, how to create an element within an element that wasn't already hardcoded into the HTML itself), and I also came away with a deeper understanding of flexbox.
